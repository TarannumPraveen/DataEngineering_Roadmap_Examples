Installation
PowerShell (Admin)
# Install Microsoft OpenJDK 11 (auto-sets JAVA_HOME!) PS> winget install Microsoft.OpenJDK.11 # ✅ VERIFY (restart PowerShell!) PS> java -version # → openjdk version "11.0.x" (Microsoft) PS> echo $env:JAVA_HOME # Install PySpark PS> pip install pyspark==3.5.1 # Hadoop setup PS> mkdir C:\hadoop\bin # Copy winutils.exe → C:\hadoop\bin\ PS> [Environment]::SetEnvironmentVariable("HADOOP_HOME", "C:\hadoop", "Machine") PS> [Environment]::SetEnvironmentVariable("Path", $env:Path + ";C:\hadoop\bin", "Machine")

# You can also setup the environment variables through the window option as shown in the video # ✅ FINAL VERIFY PS> pyspark --version
